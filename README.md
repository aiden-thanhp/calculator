## Installation

1. Clone the repo
   ```sh
   git clone https://github.com/aiden-thanhp/decoder-ring.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Start the local server for the project
   ```sh
   npm start
   ```
